HYDRA

Emergent Game / Final Year Project

HYDRA is an emergent game where the player controls thousands of agents through drawing. The player guides blue agents towards food while avoiding enemies.

My Role

  • Game Design
  • Programming
  • Art
  • UI/UX Design

Design Focus

I focused on emergent behaviour, dynamic difficulty, replayability, and player guidance. The game was inspired by computer algorithms and behavioural psychology.

Key Features

  • Player controls thousands of agents instead of one character
  • Drawing-based interaction system
  • Enemy amount changes depending on the number of player agents
  • Different patterns appear based on player behaviour

Recognition

HYDRA received an Honourable Mention for Game Design Excellence at GameBridge Festival in Cambridge, UK.
